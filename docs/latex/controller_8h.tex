\doxysubsection{Detailed Description}
Définition de l\textquotesingle{}interface de commande (Controller). 

Ce fichier centralise toutes les actions possibles que le joueur peut effectuer. Il sert d\textquotesingle{}interface normalisée entre les entrées physiques et la logique du jeu. Cette structure garantit une grande évolutivité \+: elle permet d\textquotesingle{}ajouter facilement le support d\textquotesingle{}une manette (Gamepad) ou de reconfigurer les touches sans jamais avoir à modifier le code du Modèle.
