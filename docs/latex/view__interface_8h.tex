\doxysubsection{Detailed Description}
Définition de l\textquotesingle{}interface graphique abstraite (V-\/\+Table). 

Ce fichier définit le "{}contrat"{} que toutes les Vues (SDL, Ncurses, Open\+GL...) doivent respecter. Il utilise des pointeurs de fonctions pour simuler le polymorphisme et le pattern "{}\+Strategy"{} en C pur.

Grâce à cette structure, le moteur du jeu (\doxylink{main_8c}{main.\+c}) ne sait pas quelle librairie est utilisée \+: il appelle simplement {\ttfamily view.\+render()} et la magie opère.
