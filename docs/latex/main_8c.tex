\doxysubsection{Detailed Description}
Point d\textquotesingle{}entrée principal (Main Entry Point) du jeu. 

Ce fichier orchestre l\textquotesingle{}ensemble du projet \+:
\begin{DoxyEnumerate}
\item Il lit les arguments CLI pour choisir le moteur graphique (SDL ou Ncurses).
\item Il initialise le Modèle (Données) et la Vue (Affichage).
\item Il exécute la "{}\+Game Loop"{} (Boucle de jeu) qui gère le temps, les inputs et le rendu. 
\end{DoxyEnumerate}
