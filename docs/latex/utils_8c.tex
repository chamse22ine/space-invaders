\doxysubsection{Detailed Description}
Implémentation des fonctions utilitaires (Version POSIX/\+Linux). 

Ce fichier implémente les outils de gestion du temps et d\textquotesingle{}aléatoire. Il utilise les API standards POSIX ({\ttfamily clock\+\_\+gettime}, {\ttfamily nanosleep}) pour garantir une précision à la nanoseconde, nécessaire pour une boucle de jeu fluide (60 FPS).
