Implémentation de la Vue en mode Texte (Ncurses/\+ASCII). 

Ce fichier expose l\textquotesingle{}implémentation "{}\+Low-\/\+Tech"{} du jeu. Il permet d\textquotesingle{}exécuter le jeu directement dans un terminal, en utilisant des caractères ASCII pour représenter les entités (ex\+: "{}\+A"{} pour un alien, "{}\texorpdfstring{$^\wedge$}{\string^}"{} pour le joueur). C\textquotesingle{}est une alternative légère à la version SDL.
