\doxysubsection{Detailed Description}
Implémentation Concrète de la Vue Graphique (SDL3). 

Ce module expose l\textquotesingle{}instance de l\textquotesingle{}interface utilisant la {\bfseries{Simple Direct\+Media Layer (SDL3)}}. C\textquotesingle{}est ici que sont définies les structures contenant les "{}vrais"{} pointeurs vers les fenêtres, textures et sons.

Le fichier regroupe aussi tous les chemins d\textquotesingle{}accès aux assets (Images/\+Sons).
