Detailed Description

Implémentation Concrète de la Vue Graphique (SDL3).

Ce module expose l'instance de l'interface utilisant la Simple DirectMedia Layer (SDL3). C'est ici que sont définies les structures contenant les "vrais" pointeurs vers les fenêtres, textures et sons.

Le fichier regroupe aussi tous les chemins d'accès aux assets (Images/Sons).
