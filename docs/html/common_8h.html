Detailed Description

Configuration globale et constantes du jeu.

Ce fichier définit les dimensions logiques (indépendantes de l'écran), les limites de mémoire (tableaux statiques) et la gestion du temps (FPS). Il est inclus par le Modèle, la Vue et le Contrôleur.
